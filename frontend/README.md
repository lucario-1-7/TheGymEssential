# Frontend

The TheGymEssential app: React 19 + Vite, Tailwind + shadcn/ui, Dexie (IndexedDB)
on-device store, Capacitor for Android. See the [root README](../README.md) for
the project overview, setup, Drive backup configuration, and the Android build.

```bash
npm install
npm run dev      # http://localhost:5173
npm run build
```
